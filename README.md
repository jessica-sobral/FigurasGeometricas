# Figuras Geométricas

Aplicação console para cálculo de área e perímetro de figuras geométricas.

- Quadrado;
- Retângulo.

## Funcionalidades

- Calcula a área;
- Calcula o perímetro;
- Informa os valores ao usuário.

## Tecnologias utilizadas

- dotnet 6.0

## Como executar

Realize o clone do repositório e execute o comando:

```
dotnet run
```
